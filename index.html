1. How does javascript work?

After executing JS code it gives the output via c++ and it uses internally JIT.When we write JS code we can see it through browser,and browser needs to convert the code into machine language.To do this browser use v8 javascript engine.It stays in chrome but for other browser like firefox it is spidermonkey, for internet explorer it is chakra.So at first it parse the JS code and then abstract the syntax tree and then generate bytecode.So after optimizing and compiling the JS code it converted into machine language and then it display on computer and finally we can see the output.

2. What does javascript event look do?

In v8 javascript engine, there have two things.One is heap and another is call stack.Heap uses for memory management and stack uses for when we call function these function store in the stack.Stack works as last in first out(LIFO),that means which item store in the last it will be out at first.So at first it finds out the global main function then it entered the function and do the work according to the function after that it pop out that function and check others.That's how javascript event look works.
